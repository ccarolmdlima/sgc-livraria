# Sistema de Gestão Comercial para Pequenos Negócios (SGC): Livraria

Sistema de Gestão Comercial para Livraria.

## Tecnologias
- Java 21
- Spring Boot
- MySQL

## Funcionalidades
- Cadastro de clientes
- Cadastro de produtos
- Registro de vendas
- Autenticação JWT

## Como executar
1. Criar banco MySQL
2. Rodar script SQL
3. Executar aplicação
